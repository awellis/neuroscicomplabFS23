Cue-Bedingungsvariable

Nun brauchen wir eine Variable, die angibt, ob die Bedingung ‚Äúneutral‚Äù, ‚Äúvalid‚Äù oder ‚Äúinvalid‚Äù ist. Wir erstellen eine neue Variable condition und f√ºllen sie mit case_when() mit den Werten ‚Äúneutral‚Äù, ‚Äúvalid‚Äù oder ‚Äúinvalid‚Äù. case_when() erlaubt, mehrere if_else()-Bedingungen zu kombinieren. So wird hier der Variablen condition der Wert neutral zugewiesen, wenn cue == "none" ist. Falls cue == direction ist, wird der Wert valid zugewiesen. Falls cue != direction ist, wird der Wert invalid zugewiesen.

data <- data |>
    mutate(condition = case_when(cue == "none" ~ "neutral",
                                 cue == direction ~ "valid",
                                 cue != direction ~ "invalid"))
